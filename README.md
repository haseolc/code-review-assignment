# Code Review Assignment

Amazon Bedrock Generative AI 실무 4장 미션 제출용 샘플 프로젝트입니다.

## 프로젝트 개요

이 프로젝트는 SQLite 기반 사용자 관리 CLI 프로그램입니다. 사용자 등록, 로그인, 조회, 검색, 백업, 리포트 생성 기능을 포함합니다.

## 제출 파일

- `user_manager.py`: 분석 대상 Python 코드
- `project_review_report.md`: 스타일 검사 및 보안 검사 결과 리포트

## 실행 방법

```bash
python user_manager.py
```

## 과제 기준

- Python 프로젝트 코드 200줄 이상
- 스타일 검사 수행
- 보안 취약점 검사 수행
- Markdown 리포트 생성
